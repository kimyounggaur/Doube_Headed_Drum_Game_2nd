# 조선의 북소리: 전장의 지휘자

장구 리듬을 군령으로 해석한 Vite + React + TypeScript 웹 프로토타입입니다.

## 실행 방법

```bash
npm install
npm run dev -- --port 5173
```

브라우저에서 `http://127.0.0.1:5173/`을 엽니다.

## 플레이 방법

- 왼쪽 강타: `궁` (`F` 또는 `A`)
- 왼쪽 약타: `구` (`D` 또는 `S`)
- 오른쪽 강타: `따` (`J` 또는 `L`)
- 오른쪽 약타: `다` (`K` 또는 `;`)
- 양쪽 강타: `덩` (`Space`, 또는 좌우 강타 동시 입력)
- 양쪽 약타: `더` (`Enter`, 또는 좌우 약타 동시 입력)
- 정확한 장단은 사기와 전황을 올리고, 정해진 패턴은 진법을 발동합니다.
- 오디오가 밀리면 설정에서 지연 보정을 조절하세요.

## 구현된 기능

- 캠페인 1~5장: 부산진 전투, 보초 근무, 왜구의 기습, 한산도 대첩 베타, 행주 대첩 베타
- 리듬 판정: perfect/great/good/miss, 오프셋/판정 범위/노트 속도 설정
- 전투 연동: 구음별 군령, 전황 게이지, 체력, 사기, 적 압박, 경보 게이지
- 진법: 일자진, 학익진, 장사진, 원진 패턴 감지와 전투 효과
- 장단 훈련: 기본 타법, 별달거리, 휘모리, 자진모리
- RPG 성장: XP, 계급, 장비 해금과 장착 효과
- Web Audio 합성 타격음과 효과음, 첫 클릭 후 오디오 시작
- localStorage 저장, 도감, 설정, 디버그 패널(`?debug=1`)

## 주요 파일

- `src/data/*`: 구음, 차트, 레벨, 진법, 장비, 도감 데이터
- `src/game/rhythm/*`: 판정, 차트 진행, 패턴 감지
- `src/game/battle/*`: 전투 상태, 군령 효과, 적 이벤트, 승패 판정
- `src/game/render/*`: Canvas 2D 전장과 노트 렌더링
- `src/screens/*`: 메뉴, 캠페인, 전투, 훈련, 도감, 장비, 설정, 결과 화면

## 검증
